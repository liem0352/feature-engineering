# Python 特征工程实践

> 作者：liem

## 项目简介

本项目是 Python 机器学习中特征工程的系列练习，涵盖特征编码、特征缩放、特征选择、特征离散化及缺失值处理等核心技术。

## 功能模块

- `特征编码.py`：Label Encoding、One-Hot Encoding 等编码方式
- `特征缩放.py`：StandardScaler、MinMaxScaler 等标准化方法
- `特征选择.py`：方差选择、卡方检验、互信息法等选择策略
- `特征离散化.py`：等宽/等频分箱、KMeans 聚类离散化
- `missing_values.py`：缺失值检测与处理
- `sklearn_imputation.py`：使用 sklearn 进行缺失值填充
- `exercise.py`：综合练习

## 技术栈

- Python 3
- pandas
- scikit-learn
- numpy

## 使用方式

```bash
python 特征编码.py
python 特征缩放.py
python 特征选择.py
```
